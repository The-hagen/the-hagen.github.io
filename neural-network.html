Stock Return Prediction Models

Comparing ML models for one-month forward return prediction on Oslo-listed equities, tracked live against the OSEBX benchmark.

Project Overview

This project develops and evaluates three machine learning models — a Multilayer Feedforward Neural Network, LightGBM, and Random Forest — to predict one-month forward returns for equities listed on the Oslo Stock Exchange (OSEBX).

Each model is trained on a broad cross-section of predictors spanning market microstructure, industry signals, macroeconomic indicators, and behavioural factors such as analyst estimate revisions and earnings surprises.

Monthly predictions are used to rank stocks and construct long-only equal-weight portfolios for each model. The chart below compares cumulative out-of-sample returns for each approach against the OSEBX index over the selected horizon.

Methodology

Key design decisions across all three models

📐

Feature Engineering

Market factors (momentum, volatility, beta), sector signals, macroeconomic indicators (interest rates, inflation), and behavioural variables (analyst revisions, earnings surprises).

🧠

Neural Network

Multilayer feedforward architecture with dropout regularisation and batch normalisation. Hyperparameters tuned via walk-forward cross-validation to prevent look-ahead bias.

🌿

LightGBM

Gradient boosted decision trees optimised for tabular financial data. Leaf-wise growth with L1/L2 regularisation to prevent overfitting on noisy return signals.

🌲

Random Forest

Bagged ensemble of decision trees. Naturally handles non-linear interactions and provides feature importance rankings for interpretability of signal sources.

📅

Training Scheme

Expanding-window walk-forward validation — models are retrained monthly on all available history, then evaluated on the following month's out-of-sample returns.

📊

Portfolio Construction

Each model's predicted return rank selects the top-N stocks into a long-only equal-weight portfolio, rebalanced monthly. Performance measured against the OSEBX index.
